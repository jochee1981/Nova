# New Star - 새로운 시작

신제품 및 신기술 홍보용 단일 페이지 웹 애플리케이션

## 📋 프로젝트 개요

"새로운 시작(New Star)"을 주제로 한 기업 홍보용 웹사이트입니다. 단일 HTML 파일로 구성되어 있어 별도의 서버 설정 없이 바로 사용할 수 있습니다.

## ✨ 주요 기능

- **단일 페이지 웹앱**: 모든 콘텐츠가 하나의 페이지에 통합
- **반응형 디자인**: 데스크톱, 태블릿, 모바일 모든 기기 지원
- **부드러운 스크롤 네비게이션**: 앵커 링크 클릭 시 부드럽게 이동
- **인터랙티브 UI**: 호버 효과 및 애니메이션
- **배경 이미지**: 50% 투명도로 적용된 기업 이미지 배경

## 🎨 주요 섹션

1. **히어로 섹션**: 메인 타이틀 및 CTA 버튼
2. **주요 특징**: 3개의 특징 카드 (혁신적인 성능, 스마트한 솔루션, 글로벌 표준)
3. **핵심 기술**: 4개의 기술 항목 (AI 기반, 클라우드 네이티브, 실시간 처리, 보안 강화)
4. **제품 소개**: 제품 설명 및 정보
5. **연락처**: 이메일, 전화, 주소 정보

## 🛠 기술 스택

- **HTML5**: 시맨틱 마크업
- **CSS3**: 
  - Flexbox 및 Grid 레이아웃
  - CSS 애니메이션
  - 미디어 쿼리 (반응형 디자인)
- **JavaScript (Vanilla)**: 
  - 부드러운 스크롤 기능
  - 스크롤 이벤트 처리

## 🎨 디자인 특징

- **색상**: 보라색 그라데이션 (#667eea → #764ba2)
- **배경**: 50% 투명도 기업 이미지 배경
- **스타일**: 모던한 카드 기반 레이아웃
- **애니메이션**: 페이드인 효과 및 호버 애니메이션

## 📁 파일 구조

```
Nova/
├── idex.html          # 메인 웹사이트 파일
├── README.md          # 프로젝트 설명서
└── Report/
    └── 작업결과.md    # 상세 작업 보고서
```

## 🚀 사용 방법

1. `idex.html` 파일을 웹 브라우저에서 열기
2. 별도의 서버 설정 없이 바로 실행 가능
3. 인터넷 연결 없이도 로컬에서 동작

## 🌐 브라우저 호환성

- Chrome (권장)
- Firefox
- Safari
- Edge
- 모바일 브라우저 (iOS Safari, Chrome Mobile)

- 파일명이 `idex.html`로 되어 있으나, 필요시 `index.html`로 변경하여 사용할 수 있습니다.
